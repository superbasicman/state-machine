Agent State Machine

Remote Follow Server

How to use

  1. 1. Install package:
    npm i -D agent-state-machine
  2. 2. Initialize your workflow:
    npx state-machine --setup my-workflow
  3. 3. Run your workflow:
  4. npx state-machine run my-workflow
  5. 4. The CLI will print a unique URL. Share it with anyone who needs to follow along or interact with the workflow.
  6. 5. Open the URL in a browser to see live workflow events and submit interaction responses.
